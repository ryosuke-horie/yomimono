# タイトル
タイトルには、一目で論点がわかるタイトルを記載する。
具体的かつ簡素な分で体言止めを避け「hogeはfugaする」のような文章にする。

# コンテキスト

意思決定が必要になった背景と検討過程を記述します。
SCQAのフォーマットを推奨します。
S: 状況（Situation） ... 現在の状況や背景を説明
C: 複雑化（Complication） ... 問題や課題を明確にする
Q: 疑問（Question） ... 問題に対する疑問や課題を提示
A: 答え（Answer） ... 問題解決のための提案や答えを提示

# 決定
コンテキストを踏まえた決定を、受動的ではなく、肯定的かつ能動的に記載。

# 影響
この決定の結果生じる影響を記載する。
決定の結果得られるメリットのほか、コンテキストで記載した対案を選択しなかった故のデメリットも記載する。
